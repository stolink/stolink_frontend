## 핵심 규칙

- **🔴 치명적, ⚠️ 경고**만 수정 대상
- **💡 제안**은 선택사항 → 자동 수정하지 않음
- **"✅ 코드 리뷰 통과"** 메시지가 있으면 추가 작업 중단

## 워크플로우

1. **리뷰 가져오기** (입력 없을 시):
   - `gh pr view --json number --jq .number`로 PR 번호 확인
   - `gh pr view [PR번호] --comments --json comments`로 코멘트 가져오기
   - "## 🤖 AI 코드 리뷰"로 시작하는 최신 코멘트 추출

2. **분석**:
   - "✅ 코드 리뷰 통과" 포함 시 → **수정 없이 완료 보고**
   - 🔴, ⚠️ 발견 시 → 코드 수정 진행
   - 💡만 있을 시 → 제안 내용만 사용자에게 안내 (자동 수정 X)

3. **수정** (🔴, ⚠️만):
   - `view_file`로 확인 후 수정
   - `npm run build`로 검증

4. **문서화**: `troubleshooting.md` 업데이트

5. **보고**: 수정 내역 또는 "수정 불필요" 상태 안내
