description: AI 코드 리뷰 결과를 처리하고 트러블슈팅 로그를 업데이트합니다. (자동 가져오기 기능 포함)
---

이 워크플로우는 AI 코드 리뷰 결과를 분석하여 코드를 수정하고 로그를 기록합니다. 입력값이 없으면 GitHub에서 자동으로 가져옵니다.

1. **리뷰 텍스트 확인**:
   - 사용자가 리뷰 내용을 제공했다면 해당 내용을 사용합니다.
   - **제공된 내용이 없다면**:
     1. `gh pr view --json number --jq .number`로 현재 PR 번호를 확인합니다.
     2. `gh pr view [PR번호] --comments --json comments`로 코멘트를 가져옵니다.
     3. "## 🤖 AI 코드 리뷰"로 시작하는 최신 코멘트를 찾아 내용을 추출합니다.

2. **분석 및 실행**:
   - 심각도(🔴, ⚠️)와 개선 제안(💡)을 분류합니다.
   - 관련 파일을 `view_file`로 확인하고 코드를 수정합니다.
   - `npm run build`로 무결성을 검증합니다.

3. **문서화**:
   - `troubleshooting.md` 파일에 수정 내역을 한글로 기록합니다.
   - 형식:

     ```markdown
     ## [YYYY-MM-DD] 리뷰 제목

     ### 1. 문제 제목

     - **심각도**: 🔴 Critical / ⚠️ Warning
     - **위치**: `파일 경로`
     - **문제**: ...
     - **해결**: ...
     - **상태**: ✅ 해결됨
     ```

4. **결과 보고**:
   - 사용자에게 어떤 리뷰를 처리했는지(자동 가져오기 여부 포함) 알립니다.
